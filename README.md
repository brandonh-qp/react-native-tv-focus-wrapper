# react-native-tv-focus-wrapper

React native tv focus wrapper

## Installation

```sh
npm install react-native-tv-focus-wrapper
```

## Usage

```js
import TvFocusWrapperView from "react-native-tv-focus-wrapper";

// ...
<TvFocusWrapperView>
  <View style={{ width: 100, height: 100, backgroundColor: 'blue' }} />
</TvFocusWrapperView>
```
